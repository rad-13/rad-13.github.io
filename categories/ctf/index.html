CSAW 2018 - get_it

published on November 27, 2018
get_it è la prima sfida offerta nella categoria pwning delle per la CSAW 2018 (qualificazioni). Il binario fornito è un ELF64. Download Appena scaricato possiamo analizzarlo utilizzano rabin2, un tool molto utile al fine di ottenere informazioni su un binario. Passando il parametro -I (binary info) otteniamo il seguente output: arch x86 baddr 0x400000 binsz 6754 bintype elf bits 64 canary false sanitiz false class ELF64 crypto false endian little havecode true intrp /lib64/ld-linux-x86-64.
